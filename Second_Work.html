Our goal of this project is to improve the for sale section of Craiglist website to be more hierarchy and the information is more visualized by using color, icon and font weight that users can be quick to find out the categories and the information they need. There are two members a team working on this project. I was mainly focused on the search function, sidebar and the color of the interface.

Sitemap

I started with a sitemap to build an effective information architecture and help to interpret my design idea in a systematic view for showing the specific content at each page and indicate how each page is connected. My design decision for the sidebar is to place all the universal function such as post ads, view my favorite and about function into the sidebar, and the sidebar would be displayed on every page so that users can click and use those universal function at any pages without back and forth to find those function. In the mobile version, the sidebar will collapse on the left in order to provide maximum space for important content.

Wireframe

Our wireframe redesign is focused on the posting process, navigation options, and icon adding on categories improve the user experience. I am responsible for navigation option, filter function and help to find some relative icons for categories.

For the location filter, I used hamburger button to hide all locations instead of displaying all on top, which was able to show an essential content on the limited screen space.

I used blue color to highlight the important function (refine and sort) in the sale section and added a drop showdown of the container to create a layer of an element and established a clear hierarchy of information.

I found some relative icons and placed them on the left side of each category which was easy for users to scan navigation categories quickly.

Prototype

For the prototype, I chose blue and orange as our primary color for the interface. Orange and blue are Complementary colors, also known as contrasting colors, which is useful to draw users to a few key elements such as price, and "post an ad" button at sidebar.
