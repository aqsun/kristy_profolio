Profolio Style Guide

Navigation Button

CSS selector:

ul .nav justify-content-end

Sample code:

<ul class="nav justify-content-end">

Rendered element:

  • About
  • Projects
  • Contact

MoreAboutMe Buttons

CSS selector:

.animated fadeInUp button

Sample code:

<a class="animated fadeInUp button" href="#">More About Me</a>

Rendered element:

More About Me

Headings

CSS selector:

h1,h2

Sample code:

<h1>Hello, I am Kristy</h1> <h2>Projects</h2>

Rendered element:

Hello, I am Kristy

Hello, I am Kristy

Paragraphs

CSS selector:

p

Sample code:

<p>I am currently a student at Simon Fraser University</p>

Rendered element:

I am currently a student at Simon Fraser University
